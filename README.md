# OpenDisasm
[![Build Status](https://travis-ci.org/LightningCreations/OpenDisasm.svg?branch=master)](https://travis-ci.org/LightningCreations/OpenDisasm)

Disassembler, but it's open source! And it works with all architectures someone decides to support!

Supported targets:

| Name    | Type                | Status |
|---------|---------------------|--------|
| class   | Container/Program   | WIP    |
| COFF    | Container           | WIP    |
| COM     | Container           | WIP    |
| DEFLATE | Compression format  | WIP    |
| ELF     | Container           | WIP    |
| NBT     | Data storage        | WIP    |
| PE      | Container           | WIP    |
| SWF     | Container           | WIP    |
| x86_64  | Executed program    | WIP    |
| ZLib    | Container           | WIP    |
